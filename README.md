> **Embark on an interstellar journey to defend the galaxy!** > *Galactic Air Force* is a fast-paced, vertical scrolling 2D space shooter built from scratch using **Java** and **JOGL (Java OpenGL)**. It features intense bullet-hell mechanics, epic boss fights, and a stunning visual experience generated entirely by AI.

---

## 🎮 Gameplay & Mechanics

The game offers a classic arcade experience with modern twists. Players pilot advanced starfighters through hostile alien territories, facing waves of enemies and massive level bosses.

### 🌟 Key Features
* **Epic Boss Fights:** Unique bosses for each level with complex attack patterns (Fan-shots, Parallel streams, etc.).
* **Local Co-op Multiplayer:** Team up with a friend on the same keyboard for double the firepower.
* **Dynamic Sound System:** Immersive sound effects and background music that adapts to the game state.
* **High Score System:** Track your best runs and compete for the top spot.

### ⚔️ Abilities & Power-ups
The player is equipped with a versatile arsenal to survive the cosmic storm:
* 🛡️ **Energy Shield (Press X):** Grants temporary invulnerability against enemy fire.
* ⚡ **Laser Beam (Press Z):** Fires a high-intensity concentrated beam for massive damage.
* 💥 **Super Wave (Press Space):** Unleashes a wide-spread bullet storm to clear the screen.
* ❤️ **Health Packs:** Restore your ship's integrity during battle.

## 🛠️ Installation & Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YousefSalah123/Air-SpaceShooter-JOGL.git](https://github.com/YousefSalah123/Air-SpaceShooter-JOGL.git)
    ```
2.  **Open in IntelliJ IDEA:**
    * File -> Open -> Select project folder.
    * Wait for indexing.
3.  **Setup JOGL Libraries:**
    * Ensure `gluegen-rt.jar` and `jogl-all.jar` are added to your project dependencies.
4.  **Run:**
    * Navigate to `src/mygame/Game.java` and click Run.
